Desafios desta aula: Premiação de melhores projetos

Deseja ir além? Envie seu projeto e concorra a prêmios:

Para entender como a premiação vai funcionar, assista ao tutorial e leia os passos abaixo. Não se esqueça de conferir o regulamento da premiação.

  1. Desenvolva um Projeto Relacionado com o conteúdo da Imersão Dev com Gemini

    • O projeto deve estar relacionado ao conteúdo passado durante a Imersão Dev com Gemini, organizada pela Alura em parceria com o Google, não podendo ser a mesma base de conhecimentos criada durante as aulas.

  2. Publicação do Projeto no GitHub

    • O projeto deve ser publicado no GitHub. Se precisar de ajuda para subir seu projeto, confira o tutorial aqui.

  3. Submissão do Projeto

    • A submissão do projeto deve ser feita através deste formulário entre os dias 05/09 e 07/09, até as 23h59.

  4. Publicação e Votação no Discord

    • O projeto será publicado no Servidor do Discord da Imersão Dev com Gemini 2024 para votação pela comunidade até 08/09, às 23h59.
    • A votação acontece no canal "#Votação". Dê estrelas no projeto que você mais gostou e conecte-se com outras pessoas para incentivá-las a votar no seu projeto.

  5. Avaliação pela Banca da Alura

    • Uma banca da Alura irá avaliar os 30 projetos mais votados para eleger os 10 primeiros colocados com base nos seguintes critérios:
      1. Relevância do projeto.
      2. Criatividade.
      3. Eficácia.
      4. Apresentação.
      5. Ordem de envio (critério de desempate).

  6. Anúncio dos Vencedores

    • O ranking final será divulgado na live de encerramento no YouTube no dia 09/09 às 18h30. Também será divulgado por e-mail para os vencedores e no Guia de mergulho.
  • Para saber mais, confira o regulamento da premiação aqui!
